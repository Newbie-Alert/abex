## 프로젝트 설명

### 개요

- 건강한 라이프 스타일을 위한 웹앱

### 사용하는 것

- React
- react-query
- react-router
- styled-components
- axios
- json-server
- firebase Auth
- firebase Storage

### (구현 할) 기능

- 헬스장 검색 기능

  - 검색 된 헬스장의 평점, 리뷰 등을 보여줍니다

- 운동 튜토리얼 제공
  - Youtube API를 활용한 영상 제공
  - 원하는 운동을 검색하여 영상 시청
- 다양한 카테고리의 게시판 기능
  - 헬스뿐만 아니라 다양한 라이프 스타일 관련 커뮤니티 기능

## 실행방법

#### 패키지 설치

> npm i

#### server

> json-server --watch db.json --port 8080

#### 실행

> npm start
